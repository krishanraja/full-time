# Full Time

Daily AI-narrated football recaps. Big-5 leagues. ~60 seconds per match. One morning drop. Optional account, optional push.

## Quickstart

```bash
bun install
bun run dev
```

App at `http://localhost:5173`. Backend (database, auth, storage, cron endpoint) is provisioned through Lovable Cloud — see `docs/02-developer.md`.
